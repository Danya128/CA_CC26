# Description of the project

#### For this assignment I created an electro piano.
#### The piano has 12 keys which highlight after pressing on them, and create a melody depending on what you pressed.
#### Also, after pressing the keys, the speaker creates images of notes that go up and disappear.
#### The musical instrument includes a volume slider with which help you can change the loudness of the game.
#### On the top left side you can observe 3 buttons:
#### 1. The first button generates a random melody.
#### 2. The second button plays the guitar melody in the background.
#### 3. The last third button plays the drums melody in the background.


# Instructions for use:

#### To start playing on the instrument, you need to use your laptop keys (a, s, d, f, j, k, l, ;, w, e, i, o).
#### Use the slider on the left side to change the loudness of the instrument.
#### To turn on the background song(drums, guiter or random generated piano melody), you need to use one of the three buttons on the left side of the piano.


# How it works:

#### When you press a key on the keyboard, the program plays the corresponding sound(which was downloaded and linked to each key before), highlights the key, and creates a note animation near the speaker. The volume depends on the slider.
#### Then I applied a rule that creates and moves the note up untill it hits the certain level.
#### If random mode is turned on, the program automatically plays random keys with a small delay, highlights them, and spawns notes. To create it I used a timer responsible for generate the random click on the key every 0.3 senonds
#### Here are also buttons to turn on/off background guitar and drums. Each button changes color depending on whether it is playing.
